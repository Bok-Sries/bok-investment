# 🏆 단테기법 완전 구현 최종 리포트

## ✅ 모든 단테 핵심 기법 도입 완료

사용자가 제공한 **주식단테 게시판 기법강의** 페이지 분석을 바탕으로, 앱에 다음 5가지 핵심 기법을 **정확하게** 구현했습니다.

---

## 🎯 구현된 5가지 핵심 기법

### 1️⃣ **이평 때리기 (Moving Average Hit)**
```
약세장: 112일선 < 224일선 < 448일선 (역배열)
           ↓ [매수 신호]
강세장: 112일선 ≥ 224일선 ≥ 448일선 (정배열)
```
- **신뢰도**: 95% 이상 승률
- **특징**: 하락장 바닥을 정확하게 포착
- **조건**: 역배열 → 정배열 전환 + 현재가 > 112일선

---

### 2️⃣ **256기법 (밥그릇 패턴)**
```
1단계: 큰 하락 (15% 이상)
2단계: 횡보 (박스권 - 4% 이내)
3단계: 회복 (4~15%) ← 👈 최고의 매수 타이밍
```
- **신뢰도**: 95% 이상 승률
- **특징**: 3번 자리에서의 진입이 최고 수익률
- **원칙**: 욕심 버리고 원칙만 지키면 가능

---

### 3️⃣ **일목균합표 (Ichimoku Kinky Hyo)**
```
✓ 후행스팬 상향 돌파 → 추세 전환 확인 (가장 신뢰도 높음)
✓ 전환선 > 기준선 → 골든크로스 (매수 신호)
✓ 가격 > 구름대 → 상승 추세 진입
```
- **핵심**: 후행스팬이 모든 신호의 최종 확인
- **26일 후행**: 종가를 26일 앞으로 배치
- **신뢰도**: 매우 높음 (지연되지만 정확)

---

### 4️⃣ **농사매매 (Box Trading)**
```
지지선(최근 저점) ← 반복 매수 지점
├─ 현재가
└─ 저항선(최근 고점) ← 반복 매도 지점
```
- **원리**: 지지선/저항선 반복 거래로 수익화
- **지지선**: 현재가 ≤ 지지선 × 1.02 (매수)
- **저항선**: 현재가 ≥ 저항선 × 0.98 (매도)
- **최적 박스폭**: 4% 이내 (좁을수록 기회 많음)

---

### 5️⃣ **박스권 돌파 (Breakout Trading)**
```
저항선 돌파 + 거래량 재증가 = 추격 매수 신호
        ↓
  다시 박스로 내려옴 → 저항선에서 손절
```
- **매수**: 저항선 × 1.003 이상 + 거래량 1.05배 이상
- **손절**: 저항선으로 다시 내려오면 즉시 매도
- **특징**: 박스권 완성 후 돌파 때의 추격이 수익률 높음

## 🚀 **최종 원칙**

### 1. **욕심 버리고 원칙만 지키기**
95% 이상 승률은 **원칙을 절대 어기지 않을 때만** 가능합니다.

### 2. **손절은 선택이 아닌 필수**
기준봉 저가, 지지선, 거래량 실종 중 하나라도 나면 **즉시 매도**

### 3. **분할 매수로 리스크 분산**
1차, 2차, 3차로 나누어 진입하여 평단을 유리하게 관리

### 4. **인내심이 최고의 무기**
신호가 명확할 때까지 **기다렸다** 진입 → 대기 기간 자체가 수익

### 5. **농사매매로 수익 최대화**
박스권에서는 반복 진입으로 단기 수익 + 장기 신호 대기
