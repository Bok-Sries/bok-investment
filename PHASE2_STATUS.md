# 📊 Phase 2 리스크 분석 고도화 - 현황 분석

## ✅ **이미 구현된 리스크 지표들**

### **기본 리스크 메트릭:**
1. ✅ **일간 변동성** (dailyVol)
   - 계산: standardDeviation(dailyReturns)
   - 표시: renderRisk 함수에서 업데이트

2. ✅ **최대 낙폭** (drawdown = MDD)
   - 계산: maxDrawdown(prices) 함수
   - 표시: 일일 업데이트

3. ✅ **95% VaR**
   - 계산: percentile(returns, 5)
   - 의미: 최악의 경우 5% 손실 한계

4. ✅ **99% VaR**
   - 계산: percentile(returns, 1)
   - 의미: 극단 구간 손실 한계

5. ✅ **CVaR 95% (조건부 VaR)**
   - 계산: VaR를 넘는 손실들의 평균
   - 의미: 최악 구간의 평균 손실

### **고급 리스크 지표:**

6. ✅ **하방 변동성** (downsideVol)
   - 계산: 음수 수익률만 필터링 후 표준편차 * √252
   - 의미: 손실 방향 변동성만 반영

7. ✅ **Sharpe 비율**
   - 계산: annualReturn / annualVol
   - 의미: 전체 변동성 대비 초과수익

8. ✅ **Sortino 비율**
   - 계산: annualReturn / downsideVol
   - 의미: 하방 변동성 대비 초과수익

9. ✅ **회복 필요 수익률** (Recovery)
   - 계산: recoveryReturn(drawdown) 함수
   - 의미: MDD 이후 원금 회복에 필요한 상승률

### **리스크 등급 판정:**

10. ✅ **위험 등급** (grade)
    - 기준:
      - risk > 5.5%: "높음"
      - 2.8% < risk ≤ 5.5%: "중립"
      - risk ≤ 2.8%: "낮음"

## 💡 **권장 추가 구현**

### **가장 효과적인 3가지 개선:**

#### **1. 시간대별 변동성 분석** ⭐⭐⭐
```javascript
calculateVolatilityByPeriod(prices) {
  return {
    vol30: 최근 30일 변동성,
    vol60: 최근 60일 변동성,
    vol252: 연간 변동성,
    volTrend: 변동성이 확대/축소 중인가
  }
}
```
**효과:** 리스크가 최근에 증가하는지 감소하는지 파악 가능

#### **2. 회복 시뮬레이션** ⭐⭐⭐
```javascript
calculateRecoveryScenarios(drawdown, historicalReturn) {
  return {
    daysTo50pct: MDD 절반 회복에 필요한 일수,
    daysTo100pct: 완전 회복에 필요한 일수,
    recoveryProbability: 3개월 내 회복 확률
  }
}
```
**효과:** 최악의 경우 얼마나 기다려야 하는지 이해 가능

#### **3. 포지션 한도 자동 제시** ⭐⭐⭐
```javascript
calculatePositionLimit(riskLevel, accountSize) {
  return {
    conservative: 보수적 포지션 (2-3%)
    moderate: 중간 포지션 (3-5%)
    aggressive: 공격적 포지션 (5-8%)
    kelly: Kelly 준칙 기반 (자동 계산)
  }
}
```
**효과:** 트레이더가 정확한 진입 비중을 즉시 알 수 있음
