## 🎯 **Phase 1: AI 인사이트 고도화 (즉시 시작)**

### **추가할 지표 상세**

#### **1️⃣ ADX (Average Directional Index)**
```javascript
// 추세 강도를 0-100 범위로 표시
// ADX > 50: 강한 추세
// ADX 20-50: 중간 추세  
// ADX < 20: 약한 추세/횡보

계산:
1. +DM = 현재 고가 - 이전 고가 (양수만)
2. -DM = 이전 저가 - 현재 저가 (양수만)
3. TR = Max(고가-저가, |고가-이전종가|, |저가-이전종가|)
4. +DI = (+DM의 14일 평균) / (TR의 14일 평균) * 100
5. -DI = (-DM의 14일 평균) / (TR의 14일 평균) * 100
6. DX = |+DI - -DI| / (|+DI + -DI|) * 100
7. ADX = DX의 14일 평균
```

#### **2️⃣ Stochastic (스토캐스틱)**
```javascript
// 과매수/과매도 판단 (0-100)
// 스토캐스틱 > 80: 과매수
// 스토캐스틱 < 20: 과매도

계산:
1. LowestLow = 14일 최저가
2. HighestHigh = 14일 최고가
3. %K = (종가 - LowestLow) / (HighestHigh - LowestLow) * 100
4. %D = %K의 3일 평균 (Signal)
```

#### **3️⃣ OBV (On Balance Volume)**
```javascript
// 거래량 기반 추세 강도
// OBV 상승: 매수세 강함
// OBV 하락: 매도세 강함

계산:
- 종가 > 이전 종가: OBV += 현재 거래량
- 종가 < 이전 종가: OBV -= 현재 거래량
- 종가 = 이전 종가: OBV 변화 없음
```

#### **4️⃣ VWAP (Volume Weighted Average Price)**
```javascript
// 거래량 가중 평균가
// 현재가 > VWAP: 강세 신호
// 현재가 < VWAP: 약세 신호

계산:
VWAP = Σ(고가+저가+종가)/3 * 거래량 / Σ거래량
```

#### **5️⃣ 기본 강도 점수**
```javascript
// 0-100 점수로 종목 강도 판정
// 90-100: 매우 강함 (⭐⭐⭐⭐⭐)
// 70-89: 강함 (⭐⭐⭐⭐)
// 50-69: 중립 (⭐⭐⭐)
// 30-49: 약함 (⭐⭐)
// 0-29: 매우 약함 (⭐)

구성:
- 기술 강도 (40점)
  └─ ADX(15), Stochastic(10), RSI(10), MACD(5)
- 거래량 강도 (30점)
  └─ OBV(15), VWAP(15)
- 가격 강도 (20점)
  └─ 고점 대비 위치(10), 이평선 위치(10)
- 추가 신호 (10점)
  └─ 볼린저 밴드, 기타
```
