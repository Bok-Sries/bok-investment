# 🔍 손절 기준선 점검 리포트

## ✅ 단테기법의 손절 원칙 (원본)

**2단계: 손절은 선택이 아닌 필수**
> 기준봉 저가, 지지선, 거래량 실종 중 하나라도 나면 **즉시 매도**

**9단계: 손절선 절대 준수**
> 기준봉 저가, 지지선, 거래량 실종 중 하나라도 나면 **즉시 매도** (욕심 버리고 원칙만 지키면 95% 승률!)

## 🔧 **현재 구현 코드 (app.js 라인 2593)**

```javascript
const riskLine = Math.min(forceSupport, supportLine, longLine, latest.close - latest.atr * 0.85);
```

## ❌ **문제점 분석**

### 문제 1️⃣: 기준봉 저가 미포함 ⚠️
- **단테기법**: 기준봉 저가가 손절 기준의 첫번째
- **현재 구현**: forceSupport만 있고, baseCandle.low 미포함
- **영향**: 기준봉의 실제 최저가 기반 손절이 작동하지 않음

### 문제 2️⃣: 거래량 실종 미포함 ⚠️
- **단테기법**: "거래량 실종" = 손절 조건
- **현재 구현**: volumeDryUp 변수는 있지만 riskLine 계산에 반영 안 됨
- **영향**: 거래량이 급락해도 손절선이 변하지 않음

### 문제 3️⃣: 불필요한 항목 포함 ❌
- **longLine** (장기선 = 50일선): 단테기법 원칙에 없음
- **latest.close - latest.atr * 0.85** (ATR 기반): 단테기법에 없는 방식
- **영향**: 단테기법과 무관한 손절선이 적용되어 혼동 발생

## 🎯 **손절 신호 3가지 동작 방식**

### **현재 문제:**
- 기준봉 저가: ❌ 적용 안 됨
- 지지선: ✅ 적용됨
- 거래량 실종: ❌ 적용 안 됨

### **개선 후 예상:**

#### **1️⃣ 기준봉 저가 기반**
```
기준봉 저가: 50,000원
├─ 현재가가 50,000원 아래로 내려가면 = 손절 신호 🔴
└─ baseCandle.low로 자동 감지
```

#### **2️⃣ 지지선 기반**
```
지지선: 49,500원
├─ 현재가가 49,500원 아래로 내려가면 = 손절 신호 🔴
└─ supportLine으로 자동 감지 (현재 구현됨 ✓)
```

#### **3️⃣ 거래량 실종**
```
평균 거래량: 1,000,000주
└─ 거래량이 갑자기 700,000주 이하로 급락하면 = 손절 신호 🔴
   (현재 volumeDryUp으로 감지하지만 경고 미표시)
```
