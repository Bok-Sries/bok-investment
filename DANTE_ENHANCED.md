# 🎯 단테기법 고도화 리포트

## 📺 유튜브 분석 기반 개선사항

주식단테 채널의 매매 기법을 분석하여 앱의 단테기법 섹션을 **3가지 핵심 기법** 중심으로 고도화했습니다.

---

## 🔧 추가된 핵심 기법

### 1️⃣ **일목균형표(Ichimoku Kinky Hyo) 신호**

#### 지표 구성:
- **전환선(Conversion Line)**: 9일 고가+저가 / 2
  - 단기 추세 전환 신호
  - 색상: **핑크색 선**

- **기준선(Base Line)**: 26일 고가+저가 / 2
  - 중기 저항선 역할
  - 색상: **하늘색 선**

- **구름대(Cloud)**: 전환선 + 기준선의 중점을 26일 후행
  - 지지/저항 영역
  - 가격이 구름대 위 = 상승 추세

- **후행스팬(Lagging Span)**: 종가를 26일 앞으로 배치
  - **가장 신뢰도 높은 신호**
  - 후행스팬이 종가를 돌파 → **매수 승인 신호**
  - 색상: **점선 보라색**

#### 매매 신호:
```
✓ 후행스팬 상향 돌파 → 강력한 매수
✓ 전환선 > 기준선 (골든크로스) → 상승 신호
✓ 가격 > 구름대 → 강세 장
```

---

### 2️⃣ **이평 때리기 기법 (Moving Average Hit Technique)**

#### 원리:
하락장 끝에 이동평균선이 **역배열 → 정배열**로 전환되는 순간을 포착

#### 감지 로직:
```
1단계: 약세장 = 5일 < 10일 < 20일 (역배열)
2단계: 강세 전환 = 5일 ≥ 10일 ≥ 20일 (정배열)
3단계: 확인 = 현재가 > 5일선
```

#### 거래 기법:
- 이평선 역배열 상태에서 장기간 횡보
- **정배열로 전환 후** 현재가가 5일선 상단에 위치
- **이때가 하락장 바닥** → 분할 매수 시점

#### 승률:
- **95% 이상의 승률**을 자랑하는 단테 기법
- 욕심을 버리고 원칙을 지키면 가능

---

### 3️⃣ **256 기법 (Dante's Bowl Pattern Technique)**

#### 패턴:
밥그릇 3번 자리 패턴 - "하락 → 횡보 → 회복" 구조

```
     고점
      ╱╲         
     ╱  ╲        (1차 하락)
    ╱    ╲──┐    
   ╱      └──┐   (2차 조정 / 밥그릇 1~2번)
  ╱       ┌──┘   
 ╱       ╱    (3단계: 밥그릇 3번자리 = 매수)
╱───────╱      
저점
```

#### 신호 조건:
- 장기선 위 매수세 영역 확보
- 세력 매집봉 포착 (평균 1.12배 이상 거래량)
- 긴 윗꼬리 테스트 (매물대 확인)
- 세력 평단가 재접근
- N자 파동 준비
- 손절선 사전 확정

## 📊 개선된 단테 분석 흐름

### 8단계 플레이북:

```
1단계: 일목균합표 신호 확인
└─ 후행스팬 돌파?
└─ 구름대 상단?
└─ 골든크로스?

2단계: 이평 때리기 기법 감지
└─ 이평선 역배열 → 정배열 전환?
└─ 이평선 괴리도 확인 (작을수록 신뢰도 높음)

3단계: 매집봉 선별
└─ 거래량 1.12배 이상?
└─ 윗꼬리 28% 이상?

4단계: 세력 평단 대기
└─ 추정 평단 근처 분할 진입

5단계: 눌림목 확인
└─ 3일선 눌림 또는 돌파 후 조정?

6단계: N자 파동 확인
└─ 1차 상승량, 조정 감소, 재돌파 순서 확인

7단계: 박스권 상단 반응
└─ 저항선 돌파 + 거래량 재증가?

8단계: 손절선 준수
└─ 손절기준선 이탈 시 즉시 손절
```

## 🎯 신호 강도 평가

### 스코어 등급:
- **76점 이상**: 💚 **강한 관찰** (돌파 + 눌림 동시 충족)
- **58~75점**: 🟡 **관찰** (관찰 가능한 눌림 후보)
- **38~57점**: 🟠 **보류** (조건 일부 충족)
- **38점 미만**: ❌ **대기** (신호 미확인)

### 점수 계산:
총 12개 체크리스트 (가중치 합계: 130점)
```
이평때리기(14점) + 후행스팬(12점) + 골든크로스(12점) 
+ 장기선(12점) + 거래량(12점) + 3일선(10점) + ...
```

## ⚠️ 중요 유의사항

1. **일목균합표 후행스팬이 가장 신뢰도 높은 신호**
   - 26일 전 가격 기반이므로 과거 데이터 활용
   - 현재 추세 확인용으로 최적화

2. **이평 때리기는 하락장 바닥 포착에 특화**
   - 역배열 → 정배열 전환 순간이 핵심
   - 괴리도가 작을수록 신뢰도 높음

3. **손절선은 반드시 사전에 정하기**
   - 기준봉 저가, 지지라인, 거래량 실종
   - 한 가지라도 나오면 관찰 등급 하락

4. **실시간 가격 정보 필수**
   - API 연결 실패 시 로컬 차트로 제한
   - 실시간 신호 확인 권장
