## 📊 **구현된 5가지 모델**

### **1️⃣ 추세 추종 모델 (Trend Following)**
```javascript
calculateTrendScore(indicators)

입력: SMA, MACD, ADX
점수 범위: 0-100

판정:
- 90-100: 극도 강한 상승
- 70-89: 강한 상승
- 50-69: 약한 상승
- 30-49: 중립
- 0-29: 하락추세

로직:
✅ SMA 상승 정렬 (30점)
✅ MACD 신호 (40점)
✅ ADX 추세 강도 (30점)
```

### **2️⃣ 반전 모델 (Mean Reversion)**
```javascript
calculateMeanReversionScore(indicators)

입력: RSI, Stochastic, Bollinger Bands
점수 범위: 0-100

판정:
- 70-100: 극도 과매도 (매수 신호)
- 50-69: 과매도
- 30-49: 정상
- 0-29: 과매수 (매도 신호)

로직:
✅ RSI 과매도/과매수 (40점)
✅ Stochastic 신호 (35점)
✅ Bollinger Bands 위치 (25점)
```

### **3️⃣ 거래량 모델 (Volume)**
```javascript
(이미 Phase 1에서 구현됨: OBV, VWAP)

점수 범위: 0-100

판정:
- 70-100: 강한 매수
- 50-69: 약한 매수
- 30-49: 정상
- 0-29: 강한 매도
```

### **4️⃣ 세력 모델 (Smart Money)**
```javascript
(Phase 3에서 구현됨: 기관/외국인/공매도)

점수 범위: 0-100

판정:
- 70-100: 기관 강한 매수
- 50-69: 기관 약한 매수
- 30-49: 관중
- 0-29: 기관 매도
```

### **5️⃣ 기본가치 모델 (Valuation)**
```javascript
calculateValuationScore(ticker, fundamentals)

입력: PER, PBR, ROE, Dividend
점수 범위: 0-100 (기본 50)

판정:
- 70-100: 극저평가
- 55-69: 저평가
- 45-54: 정상가
- 30-44: 고평가
- 0-29: 극도 고평가

로직:
✅ PER 평가 (±30점)
✅ PBR 평가 (±20점)
✅ ROE 평가 (±20점)
✅ 배당금 평가 (±10점)
```
